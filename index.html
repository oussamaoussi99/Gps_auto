طلب منتج GPS Auto

✅ تم إرسال طلبك بنجاح، سنتصل بك قريبًا!

* تعليمات التهيئة: * - استبدل `WEBHOOK_URL` بعنوان نقطة استقبال (Webhook) الذي تريد إرسال الطلبات إليه. * أمثلة: Formspree endpoint, Zapier webhook, Google Apps Script URL، أو نقطة خلفية على موقعك. * - إذا أردت استقبال الطلبات عبر تيلغرام أو عبر Google Sheets، يمكنك إعداد Zapier/Make/Google Apps Script * ثم وضع رابط الـ webhook تمّ إنشاء الصفحة الجاهزة ✅ — ستجدها في اللوحة إلى جانب المحادثة بعنوان صفحة بيع Gps. نقاط مهمة سريعة: الصفحة HTML كاملة (متوافقة مع الهواتف والحواسب)، تحتوي على نموذج يجمع: الاسم الكامل، رقم الهاتف، الولاية، العنوان، وملاحظات. للتحكم بكيفية استقبال الطلبات عدّل قيمة WEBHOOK_URL في ملف الـ HTML إلى رابط الـ webhook أو endpoint الذي تفضّله (مثلاً: Formspree, Zapier, Make, Google Apps Script، أو نقطة استقبال على موقعك). إذا تفضّل استقبال الطلبات مباشرة عبر واتساب، ضع رقمك في المتغيّر WHATSAPP_NUMBER داخل الملف — زر "تواصل عبر واتساب" سيفتح نافذة محادثة مع رسالة مُسبقة التعبئة. أستطيع أيضاً: ربط النموذج بصفحة Google Sheet (أرشح Google Apps Script أو Zapier) وأعدّ لك سكربت جاهز. أو أعدّل الصفحة كي ترسل البيانات مباشرة إلى بريدك الإلكتروني (باستخدام خدمة وسيطة مثل Formspree). أي تعديل تريده — نص السعر، النصوص بالعربية/الفرنسية، إضافة حقول (مثلاً: نوع التثبيت)، أو ربط بـ Google Sheets — أخبرني وسأعدّها لك فورًا.
